**ValLanches** é uma landing page moderna e elegante para venda de salgados artesanais. Desenvolvida com foco em alta conversão e estética premium (Glassmorphism e Dark Mode), a página apresenta os produtos, combos promocionais e facilita o contato direto com o cliente via WhatsApp.

## 🚀 Funcionalidades

- **Design Premium:** Efeito Glassmorphism, UI limpa e destaques visuais com gradientes em tons de ouro e laranja.
- **Cardápio Interativo:** Grade de produtos com descrição, preço e fotos.
- **Carrinho de Compras:** Sistema completo de carrinho com persistência via `localStorage`, quantidades ajustáveis e total dinâmico.
- **Integração WhatsApp:** Finalização de pedidos diretamente pelo WhatsApp com mensagem formatada, além de botão flutuante para contato rápido.
- **Animações:** Transições CSS suaves e animações de entrada com [AOS](https://michalsnik.github.io/aos/).
- **Totalmente Responsivo:** Menu *hamburger* no mobile e adaptação perfeita a todos os tamanhos de tela.

## 🛠️ Tecnologias Utilizadas

- **Frontend:** HTML5, CSS3 Avançado (CSS Variables, Flexbox, CSS Grid).
- **JavaScript:** Vanilla JS com classes ES6 (Carrinho).
- **Tipografia:** Google Fonts (Oswald para títulos, Roboto para textos gerais).
- **Ícones:** FontAwesome 6.4.
- **Animações:** AOS (Animate On Scroll) + transições CSS `cubic-bezier`.
